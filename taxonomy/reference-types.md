# Reference Types (v0.1)

- `repo` — source repository (public or private)
- `docs` — documentation (wiki, sharepoint, markdown, etc.)
- `ticket` — jira/azure boards/servicedesk issue
- `vendor` — vendor page/support portal
- `cve` — vulnerability/advisory
- `release` — release notes page
- `build` — build artifact / pipeline run
- `other`
